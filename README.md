# test_fork

來熟悉一下git怎麼用

## 指令

- clone git repository from web
```
git clone url
```

- check branch
```
git status                  // check present branch
git checkout branch         // check specific branch
```

- add change
```
git add .                   // add all
git add file                // add file
```

- add comment
```
git commit -m "comment"
```

- push
```
git push
```
